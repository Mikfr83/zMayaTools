Deform > Mirror Weights...

Mirror deformer weights across an axis.

This is like Mirror Skin Weights for deformer and blend shape weights.

Usage

Select a mesh and activate Mirror Weights.

Select the deformer to mirror from the Deformer dropdown.

If a blendShape is selected, you can either mirror weights on the deformer itself like other deformers, or mirror the weights of an individual target by selecting it from the "Blend shape target" dropdown.

If more than one mesh is affected by the deformer, select which one to mirror from the Shape dropdown.

Select the plane to mirror across. The default of YZ is correct for most meshes.

Select which directino to mirror from. If "Positive to negative" is checked, the mesh will be mirrored from right-to-left when in YZ.

Click Apply.

Limitations

  • Only simple world space position matching is supported.
  • Only deformed meshes are supported, and not curves or NURBS surfaces.
