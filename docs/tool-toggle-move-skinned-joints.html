Move skinned joints.

Introduction

Maya's built-in "Move Skinned Joints" mode only provides a move tool, so it can't be used to rotate joints.

"Toggle Move Skinned Joints" allows using any manipulator, and remains active until disabled, allowing joints to be rotated and edited freely.

Usage

Select Skin > Toggle Move Skinned Joints from the Rigging menu set. "Move skinned joints mode active" will appear in the HUD.

Joints can now be moved and rotated freely to adjust their bind position. When finished, toggle it back off.
