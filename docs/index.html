zMayaTools

Introduction

This is a collection of small tools and utilities for Maya.

Installation

Run install.bat. This will add the zMayaTools plugins to Maya's Plug-in Manager.

zMayaUtils

Load the zMayaUtils plugin to add these helper tools:

Deform > Mirror Weights... Mirror deformer weights across an axis
Deform > Copy Weights... Copy weights between deformers
Deform > Edit Blend Shape > Retarget Blend Shapes Automatic blend shape retargetting
Deform > Edit Blend Shape > Split Blend Shape Split blend shapes into blended left and right halves
Skeleton > Eye Rig A simple eye rig
Skeleton > Validate Character Detect common problems in character meshes and skinning

Nodes

zShelfMenu Show Maya shelves as menus
zRigHandle Clean, easy to select viewport rig handles

Experimental

These nodes and tools aren't quite ready yet. They may not have documentation or a UI, and the attributes of nodes may not yet be stable.
zOBBTransform Estimate the transform applied by mesh deformations
