Animation > Key > Named Keyframes

Introduction

zKeyframeNaming allows bookmarking keyframes, and allows quick navigation.

Keyframe names can be exported as a custom EXR attribute with Arnold renders.

Usage

Open the Named Keyframes menu with Animation > Key > Named Keyframes. Click Add to add a keyframe at the current time.

Selecting frames from the list will jump to that frame.

Edit > Add Arnold Attribute will add a custom attribute to the Arnold render settings, which will be included in rendered EXR files. Arnold must already be loaded.

Character sets

The zKeyframeNaming node can be added to a character set. Named keyframes will appear as keyframes in the time slider, and if keyframes are moved, their names will follow automatically.
