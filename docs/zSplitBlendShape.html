zSplitBlendShape

Split facial blend shapes into blended left and right halves

Usage

Select "Split blend shape" from Deform > Edit Blend Shape. The following options are available:
  • "Blend shape" Select the blend shape containing the target you want to split.
  • "Blend target" The blend shape target to split.
  • "Blend distance" Select the distance to blend the split. If 0, the halves will be cut sharply along the plane. If 10, the halves will blend from one to the other for 10 units around the plane. The default of 2 is good for typical faces when in standard Maya units, giving a 2cm blend.
  • "Plane" Select the plane of symmetry. The default of YZ is correct for Maya's default Y-up.
  • "Plane origin" Set the origin for the plane of symmetry. If "Plane" is set to YZ and "Plane origin" is set to 10, the blend shape will be split across a YZ plane with X = 10.

Limitations

  • This doesn't undo properly. For now, just delete the new blend shapes.
  • Menu settings aren't saved anywhere. There are only two settings, so this isn't a big deal.
