Windows > Hide Output Window

Introduction

This hides the Maya output window, which isn't useful to most users and causes clutter when several instances of Maya are open.

This is only available on Windows.

This is no longer needed from Maya 2022 onwards.

Usage

To hide the Maya output window, select Windows > Hide Output Window. This will be saved and the output window will be hidden automatically.

To show the output window again, select Windows > Show Output Window.

Limitations

  • The output window can't be completely hidden, since Maya will un-hide it each time it outputs anything to it. To work around this, the window is removed from alt-tab and taskbar list. This means the window may be visible underneath other windows.
