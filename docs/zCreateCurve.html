Create > Curve Tools > Curve From Transforms
Module: zCreateCurve.mll

Introduction

zCreateCurve creates a curve from a list of transforms.

This gives an easier way to control curves for rigging than approaches like cluster deformers and skin clusters.

This is implemented as a native plugin for performance.

Usage

Load the zCreateCurve.mll plugin and select Create > Curve Tools > Create Curve.

Create and select three or more locators (or other transforms).

In the Create Curve panel, click Create Curve or drag the locators into the window. A curve will be created, using the locator positions as CVs. If the curves are moved, the curve will update to follow.

MMB-drag transforms in the list to change the order.

Click Select Node to view the zCreateCurve node and adjust the settings of the curve.

Double-click a transform to quickly select it.

The same transform can be listed more than once. Inserting the same transform more than once is a way to increase the influence of a particular CV.

Attributes

  • Curve Degree - The degree of the generated curves. Higher degrees will be smoother, and lower degrees will follow the transforms more closely. A degree of 1 will generate lines.

    Note that this is limited by the number of transforms attached to the curve. If the degree is too high, it will be reduced to a valid value automatically.

  • Periodic Curve - If true, generate a periodic (looping) curve.
  • Parameter Range - The parameter range for the generated curves.
  • EP Curve - If set, the transforms will be used as edit points instead of CVs. This is the same as creating a curve with the EP Curve tool. Note that EP curves have several limitations (see below).

    EP curves reach the transforms exactly, rather than using them as targets. They may be useful for some cases, but can give less controllable results.

UI

  • Create Node - Create a new zCreateCurve node.
  • Delete Node - Delete the current zCreateCurve node.
  • Select Node - Select the current zCreateCurve node.

Limitations

EP curves don't support parameterRange and will always have a parameter range of 1.
EP curves don't work for all degrees, and degree 3 is recommended.
