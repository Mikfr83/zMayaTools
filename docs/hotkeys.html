Windows > Preferences > Hotkey Editor
Module: zMayaUtils

A number of hotkeys are added when zMayaTools is loaded.

These aren't assigned to keys by default. To assign them, open Windows > Preferences > Hotkey Editor. Select "Custom Scripts" and expand zMayaTools, or select "Runtime Commands" and search for its name below, such as "zNextFrameOnTimeSlider".

Hotkeys

  • zNextFrameOnTimeSlider

    Move to the next frame, staying on the time slider.

    This differs from the normal NextFrame hotkey: if the end of the time slider is reached, the time rewinds to the beginning instead of moving to the next frame.

  • zPreviousFrameOnTimeSlider

    Move to the previous frame, staying on the time slider.

    Like NextFrameOnTimeSlider, this wraps to the end of the time slider rather than moving out of the time slider range.
