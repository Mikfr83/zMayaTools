Windows > Preferences > Hotkey Editor
Module: zMayaUtils

A number of hotkeys are added when zMayaTools is loaded.

These aren't assigned to keys by default. To assign them, open Windows > Preferences > Hotkey Editor. Select "Custom Scripts" and expand zMayaTools, or select "Runtime Commands" and search for its name below, such as "zNextFrameOnTimeSlider".

Hotkeys

  • zNextFrameOnTimeSlider
    zPreviousFrameOnTimeSlider

    Move to the next or previous frame, staying on the time slider.

    Unlike the NextFrame and PreviousFrame hotkeys, these commands wrap around the edge of the time slider.

    If "Frame stepping includes the frame after the time slider range" preference is enabled, the frame at the end of the time slider range is included.

  • zGoToMaxFrame
    zGoToMinFrame

    Go to the first or last frame on the time slider.

    zGoToMaxFrame is the same as the builtin GoToMaxFrame hotkey, with added support for "Frame stepping includes the frame after the time slider range". zGoToMinFrame is also included for convenience.

  • zPickWalkAddUp
    zPickWalkAddDown
    zPickWalkAddLeft
    zPickWalkAddRight

    Pick walk, adding to the selection instead of replacing it.

    These are useful to bind to shift-up/down/left/right.

  • zToggleWireframeSelected

    Toggle wireframe on selected on and off (disabling "wireframes on everything" if active).

  • zToggleWireframeAll

    Toggle wireframes on everything on and off (disabling "wireframes on selected" if active).
